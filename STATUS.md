# MEW Station — สถานะโปรเจกต์ (บันทึกไว้กันลืม)

อัปเดตล่าสุด: 2026-07-12

## คืออะไร
เว็บสรุปข่าว AI ทั่วโลกเป็นภาษาไทย อัปเดตอัตโนมัติทุกวัน ธีมกาแล็กซี่/อวกาศ
**เว็บจริง:** https://mewuserr.github.io/ai-news-th/
**Repo:** https://github.com/Mewuserr/ai-news-th (แยกต่างหาก ไม่เกี่ยวกับ repo อื่นของผู้ใช้)
**โฟลเดอร์ในเครื่อง:** `E:\ai-news-th`

## สถาปัตยกรรม
- **Cloud routine** (Claude Code scheduled routine, ชื่อ `ai-news-th-daily-engine`) รันทุกวันตี 5 (05:03 ICT) — ไปค้นข่าวจริง สรุปเป็นไทย เขียนไฟล์ JSON แล้ว push ขึ้น GitHub → GitHub Pages deploy อัตโนมัติ
- ทำตาม instructions ใน `scripts/engine-prompt.md`
- เว็บเป็น **static site ล้วนๆ** (`scripts/generate.py` เขียน HTML จาก JSON ใน `data/`) — ไม่มี server, ไม่มี AI ทำงานตอนคนดูเข้าเว็บ (คนอื่นใช้ได้ฟรีไม่กินโควตา)
- แจ้งเตือน Windows local: `scripts/notify_check.py` รันผ่าน Task Scheduler ("AI News TH Notify") ทุกเช้า 05:20 — ดึงจากเว็บจริง ไม่ใช่ AI
- เสียงพากย์: `scripts/audio_briefing.py` (gTTS ฟรี, มีจิงเกิ้ล + โหมด 2 พิธีกร) เรียกจาก Desktop shortcuts (.bat) — ไม่ใช่ AI เช่นกัน

## ฟีเจอร์ที่ทำเสร็จแล้ว (เยอะมาก สรุปหมวดๆ)
- หน้าเว็บ: วันนี้ (index), รายสัปดาห์, คลังข่าวย้อนหลัง (มีค้นหา), สถิติ (word cloud, leaderboard, region breakdown, week-over-week), ไทม์ไลน์ข่าวใหญ่, ควิซ, สรุปประจำเดือนแบบ Wrapped (คลิกได้, มีรายละเอียด), บันทึกไว้อ่าน
- ธีม: พื้นดำสนิท + ดาวกระพริบเคลื่อนไหว (canvas) + รูปโลกจริงจาก NASA (Blue Marble) ลอยมุมขวาบน + ดวงจันทร์จางๆ มุมซ้ายล่าง + สี ฟ้า(#4fd1ff)/ชมพู(#ff4fd8) + ฟอนต์ Leelawadee UI (ไทย-อังกฤษสมดุล) + warp-speed transition ตอนเปลี่ยนหน้า
- **การ์ดแชมป์** หน้าแรก: Top 3 บริษัทจากข่าวใหญ่จริง มีคลาสตัวละคร (นักบุกเบิก/นักปราชญ์/ผู้พิทักษ์ ฯลฯ) — **กำลังคุยเรื่องเพิ่มแอนิเมชันฟันดาบ/เวทมนตร์ตอนโหลดหน้า (task ค้างอยู่ รอคุยดีไซน์กับผู้ใช้ก่อนทำ)**
- ฟีเจอร์ผู้ใช้: บุ๊กมาร์ก, ไลก์/ไม่ชอบ, ทำเครื่องหมายอ่านแล้ว, แชร์, แปลอังกฤษ, ฟังทีละข่าว/ฟังบันทึกไว้ทั้งหมด/ฟัง 2 พิธีกร (ในเว็บ ใช้ Web Speech API), ติดตามเฉพาะบริษัท, สุ่มข่าวเก่า, อ่านเร็ว 60 วิ, โหมดเล่าเป็นเรื่องราว (รอ AI เขียนรอบข่าวจริงถัดไป), Export ปฏิทิน .ics, QR โค้ด, เลือกสีธีม, ระบบยศ/สตรีค

## บั๊กที่เพิ่งแก้ (2026-07-12)
- เสียงพูดในเว็บ (Web Speech API) เคยพูดเป็นอังกฤษถ้าเครื่องไม่มีเสียงไทย → แก้ให้เช็คก่อนพูด ถ้าไม่มีเสียงไทยจะเตือนข้อความแทน
- ปุ่ม "ฟังแบบ 2 พิธีกร" กดหยุดไม่ได้ → แก้ให้กดซ้ำเพื่อหยุดได้แล้ว

## กำลังทำอยู่ตอนนี้
- เพิ่มดาวเคราะห์จริงอื่นๆ ในพื้นหลัง (ดวงจันทร์จริง แทนที่แบบ gradient เดิม, ดาวเสาร์, ดาวยูเรนัส, ดาวอังคาร) — ใช้รูปจริงจาก NASA/Wikimedia (public domain) เหมือนโลก ไม่ใช่ vector/การ์ตูน วางไว้ขอบจอเล็กๆ ไม่บังเนื้อหา

## ค้างรอคุย (ยังไม่ทำ รอ user ตัดสินใจ)
- **แอนิเมชันฟันดาบ/เวทมนตร์** ระหว่างตัวละครแชมป์ 2 อันดับแรกตอนโหลดหน้าแรก — เสนอไอเดียไปแล้ว (วิ่งเข้าหากันแล้วปะทะเป็นแสง) รอคุยรายละเอียดกับผู้ใช้ก่อนเริ่มทำ

## ตัดทิ้งแล้ว (ไม่ทำ มีเหตุผล)
RSS feed, ตั้งเวลาแจ้งเตือนหลายรอบ, แจ้งเตือนเฉพาะคำที่สนใจ, Easter Egg, ทำนายข่าวอนาคต (เสี่ยงมั่ว), ตารางเทียบราคา/เรดาร์นับถอยหลัง/มิเตอร์อารมณ์/กราฟความสัมพันธ์บริษัท (ไม่มีข้อมูลจริง), แจ้งเตือนข่าวด่วนนอกรอบ (กินโควตาเพิ่ม), หน้า "ปะทะ" แบบเลือกเอง (แทนที่ด้วยการ์ดแชมป์อัตโนมัติ)
