## คืออะไร
เว็บสรุปข่าว AI ทั่วโลกเป็นภาษาไทย อัปเดตอัตโนมัติทุกวัน ธีมกาแล็กซี่/อวกาศ
**เว็บจริง:** https://mewuserr.github.io/ai-news-th/
**Repo:** https://github.com/Mewuserr/ai-news-th (แยกต่างหาก ไม่เกี่ยวกับ repo อื่นของผู้ใช้)
**โฟลเดอร์ในเครื่อง:** `E:\ai-news-th`

## สถาปัตยกรรม
- **Cloud routine** (Claude Code scheduled routine, ชื่อ `ai-news-th-daily-engine`) รันทุกวันตี 5 (05:03 ICT) — ไปค้นข่าวจริง สรุปเป็นไทย เขียนไฟล์ JSON แล้ว push ขึ้น GitHub → GitHub Pages deploy อัตโนมัติ
- ทำตาม instructions ใน `scripts/engine-prompt.md`
- เว็บเป็น **static site ล้วนๆ** (`scripts/generate.py` เขียน HTML จาก JSON ใน `data/`) — ไม่มี server, ไม่มี AI ทำงานตอนคนดูเข้าเว็บ (คนอื่นใช้ได้ฟรีไม่กินโควตา)
- แจ้งเตือน Windows local: `scripts/notify_check.py` รันผ่าน Task Scheduler ("AI News TH Notify") ทุกเช้า 05:20 — ดึงจากเว็บจริง ไม่ใช่ AI
- เสียงพากย์: `scripts/audio_briefing.py` (gTTS ฟรี, มีจิงเกิ้ล + โหมด 2 พิธีกร) เรียกจาก Desktop shortcuts (.bat) — ไม่ใช่ AI เช่นกัน

## ฟีเจอร์ที่ทำเสร็จแล้ว (ครบตามที่คุยกันแล้วทั้งหมด)
- หน้าเว็บ: วันนี้ (index), รายสัปดาห์, คลังข่าวย้อนหลัง (มีค้นหา), สถิติ (word cloud, leaderboard, region breakdown, week-over-week), ไทม์ไลน์ข่าวใหญ่, ควิซ, สรุปประจำเดือนแบบ Wrapped (คลิกได้, มีรายละเอียด), บันทึกไว้อ่าน
- ธีม: พื้นดำสนิท + ดาวกระพริบเคลื่อนไหว (canvas) + ดาวเคราะห์จริงจาก NASA ลอยรอบจอ (โลก, ดวงจันทร์, ดวงอาทิตย์, ดาวอังคาร, ดาวเสาร์มีวงแหวน, ดาวยูเรนัส — รูปถ่ายจริง public domain ทั้งหมด ไม่ใช่ vector) + สี ฟ้า(#4fd1ff)/ชมพู(#ff4fd8) + ฟอนต์ Leelawadee UI (ไทย-อังกฤษสมดุล) + warp-speed transition ตอนเปลี่ยนหน้า
- **🚀 ยานอวกาศดวลกันแบบ Canvas บนหน้าแรก** (แทนที่การ์ดวงกลมเดิมและแทนที่ battle-intro แบบเก่าที่หายไปหลังโหลด) — วาดด้วย `<canvas>` จริง มีลำตัว ปีก กระจกห้องนักบิน เปลวไฟท้ายยาน บินหลบเคลื่อนไหวตลอดเวลา ไม่หยุด:
  - ขนาดยาน + ความถี่การยิง สะท้อน **"พลัง" จริง** (สูตร: ข่าวใหญ่×5 + ข่าวรวม) — อันดับ 1 ใหญ่สุด ยิงถี่สุด
  - ยานรองๆ ยิงเล็งไปที่ตัวที่แรงกว่าเป็นหลัก (มีเหตุผล ไม่สุ่มมั่ว)
  - มีป้ายชื่อ+เหรียญ+ตัวเลขพลังลอยกำกับข้างยานแต่ละลำในฉากเลย
  - ยังมีการ์ดข้อมูลเดิม (ชื่อ/คลาส/พลัง/รายละเอียด) อยู่ด้านล่างครบ
  - โค้ดอยู่ที่ `DOGFIGHT_SCRIPT` + `dogfight_html()` ใน `scripts/generate.py`
  - **เสร็จแล้ว ผู้ใช้อนุมัติแล้ว ("ผมโอเคแล้วคุณ") ไม่ต้องคุยต่อ**
- ฟีเจอร์ผู้ใช้: บุ๊กมาร์ก, ไลก์/ไม่ชอบ, ทำเครื่องหมายอ่านแล้ว, แชร์, แปลอังกฤษ, ฟังทีละข่าว/ฟังบันทึกไว้ทั้งหมด/ฟัง 2 พิธีกร (ในเว็บ ใช้ Web Speech API), ติดตามเฉพาะบริษัท, สุ่มข่าวเก่า, อ่านเร็ว 60 วิ, โหมดเล่าเป็นเรื่องราว (รอ AI เขียนรอบข่าวจริงถัดไปถึงจะมีข้อมูล), Export ปฏิทิน .ics, QR โค้ด, เลือกสีธีม, ระบบยศ/สตรีค
